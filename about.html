The Technology

Pure browser APIs — no server, no framework, no dependencies.

Vanilla JavaScript

Zero dependencies, zero frameworks. Pure ES6+ JavaScript for maximum speed and compatibility.

Scroll-triggered animations and lazy counter increments using the native browser Intersection Observer API.
